Trc khi chạy cần cài expo,fontawesome vào frontend\
Layout nằm ở app/screens\
Phần app(tabs) là các nút hiển thị ở thanh dưới có thể chỉnh sửa ở file app/(tabs)/_layout.tsx\
\
Lệnh cài expo: npm install (khi đã cd tới frontend)\
Lệnh cài fontawesome(trong frontend):\
npm i --save @fortawesome/react-fontawesome@latest\
npm i --save @fortawesome/fontawesome-svg-core\
npm i --save @fortawesome/free-solid-svg-icons\
npm i --save @fortawesome/free-regular-svg-icons\
npm i --save @fortawesome/free-brands-svg-icons\
